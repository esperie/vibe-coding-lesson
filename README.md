# Kailash Vibe Setup

Configuration and documentation for AI-assisted development with the Kailash SDK ecosystem.

## Overview

This repository contains setup instructions, CLAUDE.md configurations, and reference documentation for using AI coding assistants (like Claude Code) effectively with the Kailash SDK family of products.

## Contents

- **CLAUDE.md** - AI assistant configuration and directives for working with Kailash frameworks
- **sdk-users/** - User guides and reference documentation for SDK users
- **.claude/** - Claude Code configuration files

## Kailash SDK Ecosystem

This setup guide covers the following frameworks:

### Core SDK (`pip install kailash`)
Foundational building blocks for workflow automation with 110+ nodes, WorkflowBuilder, LocalRuntime, and MCP integration.

### DataFlow (`pip install kailash-dataflow`)
Zero-config database framework with automatic model-to-node generation supporting PostgreSQL, MySQL, SQLite, and MongoDB.

### Nexus (`pip install kailash-nexus`)
Multi-channel platform for deploying workflows as API + CLI + MCP simultaneously.

### Kaizen (`pip install kailash-kaizen`)
AI agent framework with signature-based programming, BaseAgent architecture, and enterprise features.

## Usage

1. Copy the CLAUDE.md file to your project root
2. Customize the directives for your specific use case
3. The AI assistant will use the specialist subagents for framework-specific guidance
